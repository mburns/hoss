Contribute

There are several ways to contribute to OSS Access Project:

1. Report a bug.

    Before submitting any bug reports or future requests, please check the bug tracker to and make sure that your bug report or future request is not listed.

2. Submit a feature request.

    Before submitting any feature requests, please check the feature request to make sure that your feature is not requested.

3. Submit a patch.

    If you find a bug and you would like to help in fixing the bug, please consider sending a patch.

4. Make a Donation.

    If you find the software to be useful, Please consider making a donation.
